# Canopy File Upload Summary

Source: https://support.getcanopy.com/en/articles/9613291-upload-a-file-to-canopy

## Upload locations (3)
- Client Files: area that collects all client files
- Internal Files: internal-only (clients cannot access)
- My Files: personal area (only the individual staff member can access)

## Common upload flow
- Go to Files from the top/global menu
- Click Add and select Upload file, or drag and drop
- The UI can vary by operating system
- After upload, the file appears on screen

## Upload to Client Files
1. Files > Client Files
2. Double click the client folder
3. Click Add
4. Select Upload file or drag and drop
5. Choose a local file

## Upload to Internal Files
1. Files > Internal Files
2. Add > Upload file or drag and drop
3. Choose a local file

## Upload to My Files
1. Files > My Files
2. Add > Upload file or drag and drop
3. Choose a local file

## File compatibility and limits
- Supported file types: .bmp .csv .doc .docx .emf .eml .heic .htm .html .jp2 .jpeg .jpg .mkv .mov .mp4 .msg .ods .odt .pdf .png .ppt .pptx .rtf .tif .tiff .txt .xls .xlsm .xlsx .xod .xps
- Max file size: 4.999342 GB
- Folder upload: folders must be zipped (uncompressed folders cannot be uploaded)

## Notes for files over 100MB
- Virus scan is not performed
- A warning appears when downloading to confirm no scan was done
- The following features are limited:
  - Preview
  - eSign
  - Annotate
  - Add pages
  - Edit in Desktop Assistant
  - Combine files
  - Use in eSign templates
- These limits apply only to newly uploaded files; existing files over 100MB are unaffected
