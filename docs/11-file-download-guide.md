# Canopy File Download Summary

Source: https://support.getcanopy.com/en/articles/9613301-download-files

## Overview
- Files can be downloaded individually or as a .zip bundle
- Downloads are available from all file storage areas in Canopy

## Download steps
1. Click Files in the top menu, then select the location
   - Client Files / Internal Files / My Files
   - If using Client Files, enter the client folder
2. Select files to download
   - Shift + click: select a range
   - Control + click: select non-contiguous files
   - Checkboxes can select files/folders
   - Full folder selection is supported
3. Right click one of the selected files
4. Choose Download from the menu
   - Files are zipped and a download progress dialog appears
